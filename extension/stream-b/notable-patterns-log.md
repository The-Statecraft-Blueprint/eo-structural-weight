# Stream B Extension: Notable Patterns Log
*Started 2026-07-06. Coder: `cowork-extension-stream-b`.*

Running log of cross-order patterns, template identifications, and diff-aware findings surfaced while coding the full-corpus extension (Eisenhower–present). Cross-referenced against `findings/notable-findings.md` (the primary coding pass's log) where a pattern extends or echoes something already named there.

## Batch 01 — Eisenhower (in progress)

**Extends an existing named narrative: Korean War mobilization centralization (EO 10161→10193→10433) gains a fourth link.** `findings/notable-findings.md` already documents progressive DPA-authority centralization across these three orders. **EO 10438** (Mar 13, 1953) is a fourth, structurally distinct data point: rather than subordinating distributed delegates under a coordinating director, it dissolves the National Security Resources Board outright and folds its Chairman's functions — including EO 10421's broad, self-amending power to redefine covered "facilities" — directly into the Director of Defense Mobilization. Recommend the existing narrative be updated to a four-order chain. Note: EO 10312, EO 10346, and EO 10421 (the three predecessor orders whose functions transfer via EO 10438) have not yet been independently scored in this stream; picking them up later would sharpen EO 10438's Flag 1 severity.

**New named sequence: Civil Service Rule VI's Schedule C → EO 10450's security-employment program (Mar 31 – Apr 27, 1953).** Diff-aware comparison of **EO 10440** against its 1947 baseline (EO 9830) found that EO 10440 creates an entirely new "Schedule C" excepted-service category and, for the first time, exempts it from ordinary Civil Service removal protections ("the Civil Service Rules and Regulations shall not apply to removals from positions listed in Schedule C") — a real weakening relative to the 1947 baseline, which extended removal protection even to competitive-status employees in excepted Schedule A/B positions. Four weeks later, **EO 10450** ("Security Requirements for Government Employment") gives individual agency heads unreviewable authority to suspend and terminate employees on an undefined "national security" standard, explicitly barring the pre-existing Loyalty Review Board from hearing further appeals (Sec. 11) and folding unrelated personal-conduct criteria — including "sexual perversion" — into the same removal mechanism as genuine espionage/sabotage concerns. Schedule C is the classification mechanism that made EO 10450's agency-head-level dismissal authority operationally unencumbered by ordinary civil-service removal procedure for the positions most likely to be targeted. Treating these as two unrelated personnel actions would miss that they form a single structural sequence enacted within one month. EO 10450 diffed against its own predecessor (EO 9835, 1947) directly — see its coding record for the specific standard-of-proof and oversight-removal language changes. This is, so far, the highest-scoring order found in this stream (65.0%), and the score is driven entirely by textually-verifiable mechanism changes (standard broadened from evidentiary to subjective; independent review eliminated; unrelated conduct criteria folded in), not by the order's historical notoriety.

**New template identified: the Railway Labor Act "Emergency Board" order.** EO 10449 (investigating a New York, Chicago & St. Louis Railroad labor dispute) is the first instance coded in this stream of a template that recurs frequently across administrations: a three-member investigatory board, an explicit conflict-of-interest bar on members, a defined reporting deadline (typically 30 days), and a symmetric, time-bound status-quo freeze on both parties pending the board's report, all operating within the Railway Labor Act's own established remedial framework. On the evidence of this first instance, the template scores cleanly (0%) — it matches the "independent, time-limited advisory bodies" positive pattern already named in `findings/notable-findings.md` (EO 9387, EO 9808), extended here from wartime/civil-rights advisory bodies to routine labor-dispute machinery. Worth tracking whether later instances ever depart from the template (an open-ended freeze, an undefined conflict-of-interest standard, or a status-quo period with no defined end) since a departure would be the interesting case, not the clean baseline.

**Recurring low-stakes template: congressional tax-return inspection grants.** EO 10435, 10442, 10447, 10451 (this batch alone) all grant a single named congressional committee inspection access to specified tax returns for a bounded purpose and bounded time. All four score 0.0% for the same structural reason: Flag 7's "direction matters" rule means *granting* Congress new oversight access never fires Preemption of Oversight, and the bounded time/purpose clears Sunset. Noting the template here so it doesn't need re-deriving on every subsequent instance across the corpus — this pattern will likely recur dozens of times across FY-bounded congressional-committee-inspection orders in later administrations too.

**Known gap flagged, not guessed:** EO 10446 supersedes EO 10410 (Nov. 1952); EO 10410 was not independently diffed against EO 10446 in this pass because the supersession is a full restatement rather than a partial amendment, and time did not permit pulling the predecessor text. Flagged explicitly in EO 10446's own coding record per the methodology's convention on undiffed predecessors — not silently assumed equivalent.

**Update to the Schedule C → EO 10450 sequence: a third link, and it's an escalation.** EO 10463 (June 25, 1953) amends Rule VI §6.4 again, this time stripping removal protection from **Schedule A** as well as Schedule C — Schedule A being the much broader "not practicable to examine competitively" category, not limited to confidential/policy-sensitive positions the way Schedule C is. Within six months of taking office, the administration went from (1) creating a removal-protection-free excepted category for sensitive positions, to (2) giving agency heads unreviewable national-security removal authority over the whole executive branch, to (3) extending the removal-protection gap to a much larger swath of the non-competitive workforce. Each order looks like a routine personnel-rule amendment read alone; the sequence is not routine.

**New divergence: international-organization loyalty review got* more* protective the same month domestic review got less.** EO 10459 (June 2, 1953) preserves and re-codifies an independent, hearing-rights-bearing "International Organizations Employees Loyalty Board" for U.S. citizens employed by the UN and other international bodies — written notice, counsel, cross-examination — just five weeks after EO 10450 barred the analogous domestic Loyalty Review Board from hearing further appeals. Two populations, two directions, same administration, same two-month window.

**Within-batch diff catch: a dual-key delegation quietly became sole authority ten weeks later.** EO 10456 (May 27) jointly delegates critical-defense-housing functions to the Secretary of Defense *and* the Director of Defense Mobilization, "who shall act jointly" — a designed two-official check. EO 10475 (July 31) revokes EO 10456 and folds the same function into the ODM Director alone. Neither order references the other's structural significance; the diff-aware convention is what surfaces it. Worth watching for whether this "joint becomes sole" pattern recurs elsewhere in the corpus — dual-key delegations may be a soft target for later quiet consolidation.

**The mobilization-centralization narrative (EO 10161→10193→10433→10438) now has a full six-order arc.** EO 10461 (June 17) formalizes ODM under Reorganization Plan No. 3 — the first step in the chain to go through the congressional-non-disapproval Reorg Plan process rather than a unilateral EO. EO 10480 (Aug. 14) then culminates it: a single order superseding thirteen prior orders (including two coded earlier in this very batch, EO 10433 and EO 10467) and vesting the ODM Director with "central programming," interagency-dispute-resolution, and general-directive authority over virtually the entire Defense Production Act program. EO 10480 is the highest-scoring order in this chunk (50.0%) and the second-highest in the batch so far, after EO 10450 (65.0%).

**Extends an existing named narrative: fair-employment enforcement's "no defined remedy" gap, in a new vehicle.** `findings/notable-findings.md` already traces EO 8802→EO 9808→EO 9981 (1941–1948): an advisory committee, "appropriate steps," no defined penalty for non-compliant employers — a gap that outlived multiple administrations and an intervening influential report. EO 10479 (Aug. 13, 1953, establishing the Government Contract Committee) re-creates the identical structural gap in a new vehicle: federal-contracting equal-employment compliance rather than direct federal employment. Contracting agencies bear "primary responsibility," the Committee can only recommend, and complaints loop back to the same agency responsible for the underlying contractor's compliance. Worth tracking whether this template (reviewing committee, complaint-forward function, no independent enforcement) recurs again as federal contract compliance mechanisms are repeatedly re-created across later administrations (this is a plausible predecessor lineage toward the eventual OFCCP-style regime).

**New highest-scoring order in the batch: EO 10491 (75.0%), directly extending EO 10450.** Six months into the term, EO 10491 amends EO 10450 to add a new removal ground: refusing, on Fifth Amendment grounds, to testify before a congressional committee about one's own alleged disloyalty. This inherits every structural problem already found in EO 10450 (unreviewable agency-head authority, no independent appeal) and adds a new one — treating the exercise of a constitutional right, rather than any adjudicated fact, as itself sufficient grounds for removal. This is the specific mechanism widely documented as used against Fifth-Amendment witnesses during congressional hearings in this era. Two amendments in one batch (this and EO 10463's Schedule A extension) both take an already-critical order and make its practical reach broader rather than narrower — worth watching whether this is a repeating shape (a facially narrow "amendment" that in practice widens an already-flagged mechanism) across the rest of the corpus.

**Full-incorporation extension: EO 10450's entire architecture ported to a new population.** EO 10497 applies "[t]he provisions of Executive Order No. 10450 ... to commissioned officers of the [Public Health] Service" — a textbook case of the scoring scheme's full-incorporation rule (inherit the parent order's flag findings for the newly-covered population). Scored at 65.0%, identical to EO 10450 itself. Worth watching for further EO 10450 full-incorporation extensions to other uniformed/quasi-military services as the corpus continues (Foreign Service, Coast and Geodetic Survey, etc. are plausible candidates given the era).

**Template contrast: Taft-Hartley "Boards of Inquiry" lack the RLA Emergency Board's self-termination.** EO 10490 (LMRA §206, maritime industry) uses the same general investigatory-board form already named as a positive pattern for Railway Labor Act boards (EO 10449), but explicitly continues "until the Board is terminated by the President" rather than expiring automatically after a fixed post-report period. Same form, different statute, one structural feature missing — a useful reminder that "looks like a template already scored clean" is not the same as "is the same template."

**EO 10501 — the classification-system foundation.** A long, substantively important order (Top Secret/Secret/Confidential, the predecessor architecture to the modern executive classification-order lineage) scores 45.0%, driven by the undefined classification standards (Flag 4, CRITICAL) and the fact that declassification review is performed by the same agencies that classify in the first place, with only NSC-level rather than independent oversight (Flags 2, 6, 8). Predecessor EO 10290 was not independently diffed against this order in this pass — flagged as a gap rather than assumed equivalent, consistent with the diff-aware convention.

**Scope creep, directly continuing the EO 10446 finding.** EO 10519 (Mar. 1954) supersedes both EO 10387 and EO 10446 (the escapee-program-specific procurement exemption logged in chunk 1). What started in EO 10446 as an exemption from three procurement/fiscal-control statutes for one narrow program (State Department's refugee "escapee program") is now, eleven months later, an exemption from eight-plus statutes covering nearly the entire Mutual Security Act. Same underlying incentive (speed over fiscal control) already flagged in EO 10446, now scoped to a much larger program. Worth watching for this shape generally: a narrow, justified-sounding carve-out created for one program, then consolidated and broadened when a later order "cleans up" by superseding it and its siblings.

**A new shape: tightening a general rule while carving out individualized discretion.** EO 10515 (Jan. 1954) moves a Colonel-retirement-suspension deadline three years earlier — a real tightening — but in the same sentence gives the Secretary of the Army unreviewable, criterion-free discretion to individually select up to twenty officers for retention under the old, later deadline. Neither "the rule got stricter" nor "the rule got a loophole" fully describes this; it's both at once, in a single amendment. Worth a name if it recurs: call it provisionally the "tighten-and-carve" pattern.

**A genuine positive-pattern contrast: not every standing advisory body lacks real periodic reauthorization.** EO 10541 (June 1954) is at least the second recorded extension of the Quetico-Superior Committee (created 1934) — a real, functioning periodic-extension requirement forcing an affirmative presidential act every few years. Contrast with EO 10472 (this batch), the National Agricultural Advisory Commission, which has no end date at all. The underlying mechanism (a one-sentence extension order) is trivially easy either way; whether a body actually gets a real periodic checkpoint or a standing indefinite life appears to depend entirely on how the original creating order was drafted, not on any external constraint.

**Bypass-language ("notwithstanding") scrutiny catches a real, if modest, finding in a low-stakes order.** EO 10545 (July 1954, Army Ration amendment) grants the Secretary of the Army open-ended, undefined authority to deviate from the prescribed ration "notwithstanding" the rest of the order, with no review or expiration requirement. Low apparent stakes (food rations) could tempt a reflexive clean score; the "notwithstanding" marker named in `scoring-scheme.md`'s Known Limitations is what surfaced the finding instead.

**Diff-aware, positive: EO 10546 fixes the exact defect EO 10465 (this batch) was flagged for.** EO 10465 tied its Marine Corps officer-grade suspension's duration to the still-open-ended Proclamation 2914 national emergency. Its successor, EO 10546 (July 1954), sets a fixed calendar end date (June 30, 1957) instead — a real, verifiable improvement in reviewability. Not every amendment in this corpus makes things worse; worth keeping this on the record alongside "tighten-and-carve" and "scope creep" as a reminder the methodology surfaces good news as readily as bad.

**EO 10450 amendment #3 in this batch (EO 10548, Aug. 1954) broadens yet again — but EO 10550, four days later, is the first to move the other way.** EO 10548 replaces the original "adjudication of insanity, or treatment for serious mental or neurological disorder without satisfactory evidence of cure" security-removal ground with "[a]ny illness, including any mental condition...in the opinion of competent medical authority" — broadening scope from a formal legal/medical predicate to an unbounded medical opinion, no defined selection process or appeal (68.75%, second-highest score in the batch after EO 10450 itself). EO 10550 (Aug. 5, 1954), by contrast, is the first EO 10450 amendment in this batch to add oversight rather than broaden a removal ground: a new 90-day agency-to-Commission reporting requirement and immediate NSC escalation of major deficiencies. Worth tracking both directions as the corpus continues — EO 10450 amendments are not a monotonic one-way ratchet, even though the broadening instances (EO 10491, EO 10548) currently outnumber the narrowing one (EO 10550) three to one within this batch alone.

**ODM-accumulation narrative reaches a seventh, then eighth, link — including a function the President had personally declined to delegate.** EO 10539 (June 1954) specifically retained the Abaca/rubber/tin acreage-direction function with the President rather than delegating it to anyone. EO 10553, two months later, delegates that very function to the ODM Director — the seventh data point in the running EO 10161→10193→10433→10438→10461→10480 narrative, and notable because even a function the President had just declined to delegate at all got folded into ODM within one administration's ordinary business. EO 10574 (Nov. 1954) extends the same narrative to an eighth link, broadening the roster of "guaranteeing agencies" bound by ODM Director certification under EO 10480 to include the Atomic Energy Commission, Commerce, Interior, and Agriculture.

**Major structural landmark: EO 10577 rewrites the entire federal appointment system and closes out EO 10180 — the same order EO 10556 had already partially overridden.** EO 10556 (Sept. 1, 1954) authorized permanent promotions/reassignments "notwithstanding" §1(b) of EO 10180 (a Korean War-era rule defaulting all promotions/reassignments to nonpermanent status, open-endedly tied to the President's own future finding that it was "no longer necessary in the interest of the national defense" — checked directly against EO 10180's text via diff-aware lookup rather than assumed). Three weeks later, EO 10577 revokes EO 10180 in its entirety, alongside the Schedule A/B/C predecessor orders (EO 9830 Part II, EO 9973, EO 10440, EO 10463) already tracked in this log's Schedule C → EO 10450 sequence, and replaces the wartime "indefinite appointment" system with the career/career-conditional appointment system that would structure federal hiring for decades. Both EO 10556 and EO 10577 read, alone, as narrow personnel housekeeping; read together with the EO 10180 text itself, they're the deliberate, dated close-out of a four-year-old emergency-era personnel regime — a genuinely positive finding, and a good example of why the diff-aware predecessor-lookup convention exists.

**A sub-component of "the emergency" gets a real, dated close-out, even while the larger emergency persists.** EO 10585 (Jan. 1, 1955) affirmatively terminates the Korean combat-zone tax-exclusion period (IRC §112(c)(3)) effective Jan. 31, 1955 — a specific, real end date — even though the broader Korean War national emergency (Proclamation 2914) remained legally open for years afterward, as this batch's EO 10465/EO 10546 pair already documents. Useful corrective to any assumption that "the emergency" was a single monolithic status during this period: some legal sub-components were affirmatively closed on schedule while others were left open indefinitely, sometimes in the very same week's batch of orders.

**Bypass-language scrutiny in a major order: EO 10575's self-certified funds-without-regard-to-law standard.** EO 10575 (Nov. 1954, Administration of Foreign-Aid Functions) is otherwise a carefully checked, multi-agency functional-assignment order (explicit inter-agency appeal/referral mechanism at §202; key functions reserved to the President at §107). But §106(b) authorizes using funds "without regard to the existing laws governing the obligation and expenditure of Government funds," limited only to purposes the exercising officer himself "deem[s]" appropriate — a self-certified standard with no textual external check. Caught by applying the "notwithstanding"/"without regard to" bypass-language scrutiny to an order that otherwise reads as well-designed, rather than assuming a well-designed order has no such provisions.

**A new, recurring low-stakes template: the "without the approval, ratification, or other action of the President" ministerial-approval-removal clause.** First seen this batch in EO 10536 (property-conveyance approvals), it recurs in EO 10583(b) (postage-stamp regulation approval), EO 10592 (school-construction-assistance finding), EO 10593 (ODM defense-housing designations), and EO 10609 (Caribbean Commission appointments) — five instances by EO 10609. Each instance scores Flag 7 PRESENT (an existing presidential check is removed) but stays low-stakes given the narrow, ministerial nature of what's being delegated. Worth continuing to track as a named template rather than re-deriving each time, and worth watching for an instance where the underlying function is *not* low-stakes.

**Fair-employment enforcement narrative gains a fourth link, in a third vehicle.** EO 10590 (Jan. 1955) establishes the President's Committee on Government Employment Policy, superseding EO 9980's Fair Employment Board — the direct-federal-employment counterpart to EO 10479's federal-contracting version of the same structural gap (`findings/notable-findings.md`, EO 8802→EO 9808→EO 9981). Fourteen years and at least three total reorganizations after EO 8802, the same shape persists: an advisory committee, no defined remedy, and a complaint process that loops back to the same agency whose own conduct is being reviewed.

**A tenth ODM data point comes with a genuinely careful boundary-drawing example.** EO 10601 (Mar. 1955, Commodity Set-Aside) continues routing strategic-materials programs through ODM Director certification (per the running EO 10161→...→10574 narrative) but is also a clean instance of *partial*, not full, incorporation-by-reference: it extends only Part II (diplomatic-mission coordination) of EO 10575, not that order's full architecture, so EO 10575's own §106(b) bypass-authority finding is correctly not inherited. Also notable for §1(b)'s explicit exclusion of national-emergency-declaration authority from an otherwise broad delegation — a deliberate, textually visible limit, not an assumed one.

**Two more "normalization of extraordinary authority" orders, joining EO 10577 and EO 10585.** EO 10608 (May 1955) abolishes the U.S. High Commissioner position in Germany, replacing occupation-era "supreme authority" with an ordinary bifurcated diplomatic/military structure as the Bonn-Paris Conventions take effect. Combined with EO 10577 (civil-service system normalization, closing out EO 10180) and EO 10585 (Korean combat-zone tax close-out), this batch now has three clear examples of the Eisenhower administration formally and deliberately winding down wartime/occupation/emergency-era extraordinary authority via ordinary EOs rather than letting it lapse implicitly or persist indefinitely — a pattern worth naming and watching for whether it continues as the batch moves further from the Korean War/WWII-occupation period.

**Rapid institutional churn in foreign-aid administration.** EO 10476 (Aug. 1953) → EO 10575 (Nov. 1954) → EO 10610 (May 1955): three total reorganizations of the same basic foreign-aid administrative function (Foreign Operations Administration, then folded into a new International Cooperation Administration within the State Department) within about 21 months. Each order is individually well-designed (checked delegations, reserved presidential functions, referral mechanisms for disputes), but the sheer frequency of total reorganization in this one policy area is itself worth flagging as a pattern — whether it recurs with other Mutual Security Act-adjacent machinery as the corpus continues is worth tracking.

**A second "tighten-and-carve" instance, and it's less bounded than the first.** EO 10617 (June 1955) suspends mandatory Colonel retirement with a genuine fixed backstop (June 30, 1957) — the "tighten" half — while giving the Secretary of the Army unreviewable, criterion-free ("in the public interest") discretion to individually retain colonels past that point, with no numeric cap at all, unlike EO 10515's stated 20-officer limit. Same shape, less bounded.

**The EO 10546 fix (fixed calendar end date replacing an open-ended Proclamation 2914 tie) did not become standard practice.** Two more officer-suspension orders in the same single chunk — EO 10616 (Marine Corps brigadier generals) and EO 10632 (Navy Medical/Dental Corps) — revert to tying suspension duration solely to the still-unresolved Proclamation 2914 emergency, with no fixed backstop. Worth tracking whether any later order fixes this systemically (e.g., a general housekeeping order converting all Proc. 2914-tied suspensions to fixed dates) or whether this remains permanently ad hoc, order-by-order.

**Fourth "normalization of extraordinary wartime authority" finding: EO 10628 restores ordinary UCMJ punishment limits for Far East Command, closing out a 1951 wartime suspension (EO 10247).** Joins EO 10577 (civil-service normalization), EO 10585 (Korean combat-zone tax close-out), and EO 10608 (Germany occupation-authority wind-down) as a fourth clear instance of this administration formally and deliberately winding down extraordinary wartime-era authority via an ordinary EO, rather than letting it lapse implicitly.

**New pattern: bulk ministerial-delegation orders that bundle live, frequently-used authorities with dormant wartime powers.** EO 10637 (Sept. 1955) delegates roughly twenty distinct Coast Guard-related presidential functions to the Secretary of the Treasury "without the approval, ratification, or other action of the President" — by far the broadest instance yet of the ministerial-approval-removal template first seen at EO 10536 (eight instances total in this batch so far: EO 10536, EO 10583(b), EO 10592, EO 10593, EO 10609, EO 10613, and now EO 10637's twenty functions at once). Several of the bundled authorities are dormant WWI/interwar/WWII-era neutrality-law vessel-detention powers (1916-1941 acts) delegated undistinguished alongside live personnel functions — meaning if any of those largely-dormant powers were ever invoked, it would happen with no presidential sign-off and no visible indication anyone had revisited whether that specific delegation still made sense.

**A genuine positive counter-example: EO 10647's conflict-of-interest safeguards for WAE/consultant appointees.** Where several of this batch's findings involve self-certified or undefined standards (EO 10575's §106(b), EO 10590's advisory-only remedy gap), EO 10647 (Nov. 1955, replacing EO 10182/10205) is the strongest ethics/transparency design found so far: appointees are barred from negotiating contracts with their own private employer or acting on that employer's relief applications, face a two-year post-employment claims-prosecution bar, must publicly disclose financial interests in the Federal Register, and the CSC Chairman must audit compliance quarterly and report to both the President and the Joint Committee on Defense Production — a real congressional-reporting channel, unusual for this era's ethics structures.

**A structurally self-referential oversight body: EO 10656 establishes PFIAB's direct predecessor.** The President's Board of Consultants on Foreign Intelligence Activities reviews CIA performance and foreign-intelligence quality, but its entire reporting channel runs to the President alone — §4 bars members from revealing anything learned, even to Congress, except to the President or his designees. The body created specifically to check U.S. intelligence performance has no textual mechanism for its findings to reach anyone outside the executive branch it's reviewing. This closed-loop design would remain a defining, debated feature of PFIAB for decades.

**ODM-authority narrative reaches a fourteenth link, but the pace and scope of each new link is visibly narrowing.** EO 10638 (stockpile release authority, attack-triggered), EO 10660 (National Defense Executive Reserve), and EO 10662 (narrow DPA §304 mineral-development amendment) extend the running EO 10161→...→10637 narrative to data points twelve through fourteen, but each is narrower and more specific than the sweeping reorganizations earlier in the term (EO 10480, EO 10461) — worth watching whether this represents the narrative settling into routine maintenance as the administration's mobilization architecture matures, rather than continued net expansion.

**A bounded, one-time retirement exemption vs. a standing, undefined one — same subject, different design, six weeks apart.** EO 10674 (July 1956) exempts a specific window of officials (those reaching retirement age July-Sept. 1956) from automatic separation, extended only to a fixed date. EO 10682 (Oct. 1956) instead delegates to the Civil Service Commission a standing, ongoing authority to exempt any employee (other than Presidential appointees) from automatic separation "when, in his judgment, the public interest so requires" — no time bound, no further criteria. Worth tracking whether the standing version increasingly displaces the bounded, one-time version as the pattern recurs.

**Two well-designed, structurally parallel tax/retirement-withholding frameworks.** EO 10672 (DC income tax withholding by federal agencies) and EO 10679 (state retirement-system withholding for National Guard civilian employees), issued the same year, share an identical design: a 120-day agreement deadline, specific carve-outs for armed-forces compensation and non-resident employees, and an explicit anti-waiver clause preserving federal sovereign immunity from more burdensome treatment. Worth noting as a positive template — this era's tax/benefits-withholding coordination orders are consistently well-bounded where they appear in this batch.

**A second closed-loop national-security coordinating body: EO 10700's Operations Coordinating Board.** Reconstituted within the NSC structure (superseding EO 10483 wholesale — resolving the gap flagged when EO 10598 merely amended EO 10483's membership without independently scoring it), the OCB coordinates operational execution of approved national security policy, including, per its own text, "the particular climate of opinion the United States is seeking to achieve in the world." It reports only to the NSC, never to Congress. Paired with EO 10656 (PFIAB's predecessor, also this batch), this is now a second instance of a national-security coordinating body built with no textual outside-accountability channel — worth watching as a recurring structural type across the rest of this era's national-security apparatus.

**One order with no available text: EO 10695.** "Radio Frequencies" (Jan. 1957) remains confidential and unpublished to this day; the corpus record itself states only "[confidential - not published]." Coded with all flags NOT_APPLICABLE and a clear note, per the methodology's convention against guessing when text is genuinely unavailable rather than merely difficult to find.

**A genuinely well-bounded ODM delegation, for contrast: EO 10705's wartime radio-control authority.** The fifteenth data point in the ODM-authority narrative, but notably one of the most carefully limited: exercisable only during an actual or proclaimed war, explicitly content-neutral, redelegable only to Senate-confirmed officials, with facility-takeover decisions requiring the Director's own case-by-case approval. Useful as a comparison point against the narrative's less-bounded instances.

**Major standalone finding: EO 10713, "Providing for the Administration of the Ryukyu Islands" (June 5, 1957) — the most structurally concentrated order found in this stream to date (70%).** Establishes an entire territorial government (elected legislature, Ryukyuan Chief Executive, dual court systems) for the civilian population of Okinawa, then subordinates all of it to a U.S. military officer, the High Commissioner, who holds veto, law-annulment, and removal power over that government, plus a self-triggered clause allowing him to "assume in whole or in part, the exercise of full authority in the islands, if such assumption of authority appears mandatory for security reasons" (Sec. 11) — an undefined, self-judged standard with no outside review. Sec. 10(e) separately strips courts of jurisdiction over the U.S. Government in the territory "unless specific authority has been conferred...by the Congress of the United States" — an explicit, textual preemption of judicial oversight requiring affirmative congressional action to restore a default-absent check. This combination (broad self-judged emergency-assumption trigger + explicit court-jurisdiction bar) is more extreme than anything else found in this batch and is logged as a standalone finding rather than folded into an existing template, though it shares DNA with the vague-security-standard pattern seen in national-security coordinating bodies (EO 10656, EO 10700) and the oversight-preemption pattern seen in the ministerial-approval-removal template — here both patterns appear together, in the same order, applied to governance of an entire population.

**Ministerial-approval-removal template, instance #10 — and an escalation, not just a repetition: EO 10729, "Special Assistant to the President for Personnel Management" (Sept. 16, 1957).** Prior instances (EO 10536, 10583(b), 10592, 10593, 10609, 10613, 10637, 10661) authorized bypassing the President's own approval/ratification step for a defined set of functions. EO 10729 goes further: Sec. 3(b) delegates the power to *define the scope* of that bypass to a newly-created, non-Senate-confirmed White House aide, who may "exclude any specific matter or matters" from the consultation requirement over Civil Service Commission personnel actions and "terminate any exclusion" at will — with no criteria given for either move. The bypass mechanism itself has become a standing, self-administered discretion held by a single unconfirmed office, rather than a fixed, defined carve-out. Highest-scoring order in this chunk after EO 10713 (60%).

**Full-depth treatment: EO 10730, Little Rock/Central High School enforcement (Sept. 24, 1957).** Federalizes the Arkansas National Guard and authorizes active-duty forces to enforce a federal district court's desegregation order after Governor Faubus's National Guard deployment blocked Black students' enrollment. Structurally notable for being unusually well-anchored to independent judicial authority — the order's Sec. 1 authority derives from enforcing "orders of the United States District Court," not a self-judged presidential emergency finding — which is why it scores relatively low (20%) despite the gravity of federalizing a state's National Guard against the Governor's wishes. The genuine structural gaps are narrower: the federalization runs "for an indefinite period and until relieved by appropriate orders" with no textual sunset or review point (Flag 6), and the underlying statutory mechanism (10 U.S.C. 332-334) is broad enough that its second-order effect is to normalize domestic military deployment as an enforcement tool, available in future circumstances that might be less judicially anchored than this one (Flag 9). Worth contrasting directly against EO 10713 earlier in this same chunk: both involve significant domestic/territorial use of federal power, but 10730 is triggered by and bounded to an independent court order while 10713 is triggered by the exercising official's own undefined "security" judgment — the same raw power-concentration magnitude, arrived at through structurally opposite paths.

**Possible third instance of the "tighten-and-carve" pattern: EO 10737, disaster relief administration (Oct. 29, 1957).** Previously named for EO 10515/10617 (a bounded general rule paired with unreviewable, criterion-free individual-selection discretion in the same amendment). EO 10737 pairs a genuine one-year default sunset on federal disaster assistance (Sec. 2(f)) with an open-ended extension power reserved to the FCDA Administrator — "in unusual circumstances," or, for agricultural-purpose relief specifically, "for such purposes and such periods of time as he may determine to be necessary," neither phrase further defined. Different subject matter (disaster relief vs. personnel/registration) but the same underlying shape: a real sunset immediately paired with a standardless escape hatch.

**Fair-employment-enforcement narrative, capacity-only continuation: EO 10733 (Oct. 10, 1957).** Adds an Executive Vice Chairman and one member to the Government Contract Committee (EO 10479→10557→10590→10733). Pure staffing/leadership expansion — does not touch the complaint-and-remedy structure, so the advisory-only, no-defined-remedy gap identified at EO 10479 is preserved unchanged, neither strengthened nor weakened.

**Ministerial-approval-removal template, war-criminal-clemency thread escalates: EO 10747 (Dec. 31, 1957).** Directly continues the thread first flagged at EO 10613 (which amended the multi-member Clemency and Parole Board for War Criminals). EO 10747 abolishes that Board outright and gives the Secretary of State sole authority to accept or decline Japan's war-criminal sentence-reduction recommendations "without the approval, ratification, or other action of the President" (Sec. 1) — collapsing a board-based process into one Cabinet officer's presidential-approval-free discretion, on the same substantive subject (Japanese war-criminal clemency under the Treaty of Peace with Japan) as the earlier instance. Second-highest scoring order in this chunk (40%).

**Diff-aware confirmation using an actual validation-pass record: EO 10746 against EO 10560.** EO 10560 was one of the 20 original claude-church-bells-v1 Eisenhower records (mostly ABSENT/NOT_APPLICABLE, Flag 9 PRESENT for its historical significance as the "Food for Peace" / Public Law 480 administration order). Rather than re-deriving EO 10746's amendment from scratch, its flag findings were checked directly against EO 10560's actual coding record — confirming the amendment is a pure cross-reference update (accommodating a new 1957 Export-Import Bank loan authority) that preserves the baseline unchanged across all 11 flags. Recommend doing this direct-comparison check whenever a later amendment targets one of the 20 church-bells-v1 orders specifically, since the original record is available in the DB rather than needing to be reconstructed from memory.

**Possible discovery: a single "hub" order behind multiple ministerial-approval-removal instances.** EO 10759 amends "Executive Order No. 10530...Providing for the Performance of Certain Functions Vested in or Subject to Approval of the President" — a title that matches the template's language almost exactly. This raises the possibility that EO 10530 is the central framework order to which several previously-logged instances (EO 10536, 10583(b), 10592, 10593, 10609) are themselves amendments, rather than independent freestanding orders — which would mean the template is less "the same pattern re-invented many times" and more "one hub order repeatedly amended to add new bypass-eligible functions." Not yet verified against each prior instance; flagged for a future pass to check whether those EO numbers' own text identifies EO 10530 as the order being amended.

**Proclamation 2914 open-ended-duration pattern extends to April 1958: EO 10764.** Suspends the Eight-Hour Law for CAA laborers "until terminated or revoked by the President," invoking the December 1950 Korean War national emergency proclamation as live legal predicate nearly five years after the armistice and with no textual acknowledgment that the underlying war ended. Notably, this does *not* meet the strict Zombie Emergency Trap sub-flag definition (which requires an order to *acknowledge* lapsed authority) — instead of acknowledging staleness, the order simply proceeds as if no time had passed, which may be the more concerning variant: silent reliance on a dormant-but-technically-unexpired proclamation, rather than an argued defense of continuing to use it.

**A cleaner divergence in the same UN-Secretariat loyalty-review order: EO 10763 vs. EO 10459.** Both amend EO 10422 but touch different parts of it. EO 10459 (1953, already logged) strengthened the back-end hearing-rights structure (written notice, counsel, cross-examination) for U.S. citizens facing removal from UN Secretariat consideration. EO 10763 (1958, this chunk) broadens the front-end investigative-source list and adds an automatic-FBI-referral trigger for derogatory findings — a expansion of investigative reach that, on the text available, leaves the hearing-rights protections untouched. Two amendments to the same order, moving in different directions on different sections, five years apart.

**The ODM-authority-accumulation narrative reaches its most consequential point yet: EO 10773 (July 1, 1958), the sixteenth data point.** Implements Reorganization Plan No. 1 of 1958 by merging the Office of Defense Mobilization with the Federal Civil Defense Administration into a new "Office of Defense and Civilian Mobilization," consolidating functions previously assigned across 26+ separate prior orders under a single Director with power of further redelegation, and preserving (Sec. 8) all prior regulatory action without fresh review. Where EO 10480 (Aug. 1953, previously the narrative's high-water mark at 50%) unified the defense-mobilization program alone, EO 10773 goes further by fusing that entire apparatus with the separate civil-defense chain of authority — wartime industrial mobilization and domestic disaster/civil-defense response under one office for the first time. Scores 55.56%, now the second-highest-scoring order in this stream after EO 10713 (Ryukyu Islands, 70%) and ahead of EO 10450 (65%). Renamed just two months later by EO 10782 ("Office of Civil and Defense Mobilization") pursuant to an intervening Act of Congress — the July 1958 structure was evidently provisional.

**A well-checked exception to this batch's "without regard to law" bypass pattern: EO 10789 (Nov. 14, 1958).** Authorizes DoD and ten additional civilian agencies to contract "without regard to the provisions of law relating to the making, performance, amendment, or modification of contracts" whenever, in the relevant Secretary's own judgment, "the national defense will be facilitated thereby" — invoking the by-now-familiar stale Proclamation 2914 (Dec. 1950) predicate nearly eight years after the Korean War armistice. Unlike most bypass instances logged in this batch, however, this order carries real countervailing safeguards: a $50,000 threshold requiring Assistant-Secretary-or-above approval, a mandatory annual report to Congress itemizing contractor name/cost/description/justification for actions above that threshold, mandatory Comptroller General audit-access clauses in every contract, and an explicit list of actions NOT authorized (cost-plus-percentage-of-cost contracting, bond waivers, price increases above the lowest rejected bid). The one soft spot: the reporting and public-inspection requirements themselves carry a self-determined national-security nondisclosure exception, so the safeguards can be partially switched off by the same officials they're meant to check. Worth holding up as a comparison point against EO 10713 (Ryukyu Islands) and the ministerial-approval-removal template, neither of which has any comparable reporting/audit mechanism at all.

**A distinct, more personal exemption-architecture pattern: EO 10778, a named-individual civil-service bypass.** "Authorizing the Appointment of William K. Harris to a Competitive Position Without Compliance With the Civil Service Act and Rules" (Aug. 20, 1958) exempts one specifically named person from the entire competitive civil-service hiring framework, with no stated reason beyond "recommended by the Secretary of the Navy." Distinct from the categorical Schedule A/B/C exemption mechanisms already tracked in this batch (EO 10440, EO 10463): those create exemption classes; this exempts an individual by name, with no criteria at all. Worth watching for recurrence elsewhere in the corpus — if named-individual bypass orders like this show up repeatedly, they'd form their own template.

**Proclamation 2914 pattern gains a structural variant: EO 10780/10781 (Sept. 2, 1958) tie suspension to the emergency's end, not to presidential discretion.** Unlike EO 10764's purely open-ended "until terminated or revoked by the President," these two companion orders suspend Navy promotion-related statutes "until June 30 of the fiscal year following that in which the national emergency proclaimed by Proclamation No. 2914...shall end" — tethered to an objective external event rather than unfettered discretion. A better structural design in principle, though it proved just as durable in practice, since Proclamation 2914 was not formally terminated for decades.

**EO 10530 hub theory confirmed: EO 10790 is a second amendment to the same base order.** EO 10759 (this batch) first raised the possibility that EO 10530 ("Providing for the Performance of Certain Functions Vested in or Subject to the Approval of the President") is a repeatedly-amended framework order rather than each ministerial-approval-removal instance being independent. EO 10790 amends the same order's subsection (c) (following EO 10759's amendment to subsection (b)) — two separate, narrow, function-specific amendments to the identical base order, confirming it functions as a standing hub. Separately, EO 10813 (this chunk) reveals EO 10530's Sec. 1(h) delegation being used for an unrelated purpose (Bureau of the Budget approval of a TVA land-transfer agreement), showing the hub order's delegated authority is in active use well beyond the personnel-approval context most associated with the template.

**The EO 10729 self-administered exclusion mechanism is a reusable template, not a one-off: EO 10800 (Jan. 15, 1959).** EO 10729 (this batch) gave the Special Assistant to the President for Personnel Management standardless power to exclude any CSC personnel matter from a presidential-approval consultation requirement, and to undo that exclusion, at will. EO 10800 (implementing the Government Employees Training Act) reuses the identical clause verbatim, applied to a different statutory domain (employee training rather than personnel actions generally) — confirming the mechanism is a general-purpose device being deliberately extended across statutory contexts, not confined to its original personnel-management setting. Worth watching for further reuse of this exact language elsewhere in the corpus.

**New template spotted: the "CIA carve-out."** EO 10805 (Feb. 18, 1959) exempts the CIA from eight full sections and parts of two more of the Government Employees Training Act (issued the same month as EO 10800, which implements that same Act generally), based on the President's own "public interest" determination under a statutory except-any-agency provision. This is a comprehensive, agency-specific exemption from most of the CSC-administered oversight structure of a general personnel statute. Given the CIA's broad general statutory exemption from many personnel/administrative laws, this is likely to recur as later administrations implement new general-workforce statutes — worth tracking as its own named template across future batches.

**Diff-aware confirmation using two more actual validation-pass records: EO 10799 (against EO 10560) and EO 10807 (against EO 10521).** Both of the 20 original claude-church-bells-v1 Eisenhower records now have a second later-batch amendment checked directly against their actual coding: EO 10560 (Food for Peace) via EO 10746 and now EO 10799; EO 10521 (NSF/Federal Council for Science and Technology's predecessor) via EO 10807. Both amendments preserve their predecessor's clean baseline scores unchanged. Recommend continuing this direct-comparison approach for the remaining 18 church-bells-v1 orders whenever a later amendment targets one of them specifically.

**A meaningfully balanced appeals body, worth contrasting with several one-sided gatekeeping committees logged elsewhere: EO 10794's Canal Zone Board of Appeals.** Five members — one CSC-nominated, two selected from among employees (after consultation with employee organizations), two selected by the Secretary of the Army — deciding by majority vote. Unlike several advisory/gatekeeping bodies elsewhere in this batch composed entirely of executive-branch or agency-selected members, this body gives the reviewed population (Canal Zone employees) a real voice in its own composition. The same order, however, explicitly preserves EO 10450 and EO 10501 (this batch's most-flagged security/loyalty-removal orders) as unmodified for Canal Zone security-position determinations — a genuinely mixed record within a single order.

**EO 10530 hub theory now confirmed by a third amendment: EO 10836 (Sept. 8, 1959).** Following EO 10759 and EO 10790 (both this batch), EO 10836 is a third narrow, function-specific amendment to EO 10530 ("Providing for the Performance of Certain Functions Vested in or Subject to the Approval of the President"), adding National Capital Planning Commission and Alaska Omnibus Act functions. Three independent amendments to the same base order, each adding a discrete delegated function, is now a well-established pattern rather than a hypothesis — EO 10530 functions as a standing repository for functions the President delegates while remaining reviewable in principle.

**Best-designed instance yet of the ministerial-approval-removal template: EO 10841 (Sept. 30, 1959), nuclear-cooperation risk determinations.** Jointly delegates to the Secretary of Defense and the Atomic Energy Commission — not a single officer — the authority to determine whether proposed nuclear-cooperation arrangements pose an "unreasonable risk to the common defense and security," "without the approval, ratification, or other action of the President." Unlike most instances of this template (EO 10536 through EO 10835, all this batch), this one requires two independent agencies to agree, and Sec. 2(b) requires any disagreement between them to escalate to the President rather than defaulting to either party. A consequential subject (nuclear-materials/technology cooperation, comparable in stakes to the war-criminal-clemency thread at EO 10613/10747) handled with meaningfully more structural care than EO 10729's single-office, standardless-discretion version.

**Taft-Hartley Board of Inquiry template, sixth instance, applied to the 1959 steel strike: EO 10843.** Continues the indefinite-continuation pattern already tracked (EO 10490, 10542, 10543, 10689, 10710) — "shall continue in existence...until the Board is terminated by the President," with only the initial report deadline fixed. Notable for the scale of the underlying dispute (one of the postwar era's largest labor conflicts) rather than any structural departure from the established template.

**Two Proclamation 2914 references distinguished by character, not just count: EO 10824/10840 vs. the batch's other invocations.** These two Assignment-of-Claims-Act agency designations (NASA and FAA respectively) quote the underlying 1940 statute's text, which lists "the national emergency proclaimed December 16, 1950" as one of several alternative temporal triggers for a contract-payment-protection mechanism — but neither order itself makes a fresh presidential emergency finding the way EO 10764, EO 10780/10781, and EO 10789 do. Worth keeping this distinction sharp when tallying Proclamation 2914 references across the corpus: quoted statutory boilerplate is a materially different (and less concerning) category than an operative invocation.

**Notable reversal: EO 10859 abolishes the Career Executive Program without explanation (Feb. 5, 1960).** EO 10724 (Aug. 1957) -> EO 10758 (Mar. 1958) -> EO 10777 (Aug. 1958) had been tracked across this batch as a clean, positive example of a merit-based personnel program — explicit merit-system guardrails, political-party balance requirements, fixed reporting deadlines, and final decision authority retained by the President. EO 10859 abolishes the entire program in a single sentence, roughly eighteen months after its full implementation, with no reason given in the order's own text. A reminder that positive-scoring governance mechanisms tracked in this stream are not necessarily durable — this is the first instance in the batch of a previously-flagged clean/positive mechanism being reversed outright rather than amended, extended, or left in place.

**Major finding: EO 10865, "Safeguarding Classified Information Within Industry" (Feb. 20, 1960) — the Industrial Security Program, worth direct comparison against EO 10450.** Establishes real, substantial due-process protections for industrial personnel security-clearance determinations (written statement of reasons, sworn reply, personal appearance, counsel, cross-examination rights, written final decision) — considerably more protective than EO 10450's domestic-employee security-removal apparatus (this stream's highest-scoring order at 65%, which bars Loyalty Review Board appeals entirely). Sec. 7 even explicitly clarifies that adverse determinations are "in no sense...a determination as to the loyalty of the applicant," a protective distinction EO 10450 does not draw. But EO 10865 shares EO 10450's core structural flaw: Sec. 9 preserves a conclusive, non-delegable department-head override — "if the security of the nation so requires" — that can set aside the entire due-process regime whenever the department head himself determines the ordinary procedures "cannot be invoked consistently with national security," with no external check on that determination. Same escape valve, more protective baseline. Scores 30%, the second-highest in this chunk.

**EO 10530 hub theory: fourth confirmed amendment (EO 10852, Nov. 27, 1959).** A narrow terminology update, following EO 10759, EO 10790, and EO 10836 (all this batch). Four independent amendments to the same base order over roughly fifteen months is now simply how EO 10530 operates — a standing repository for delegated, presidentially-reviewable functions, added to piecemeal as new statutes require.

**EO 10530 hub theory: fifth confirmed amendment (EO 10889, Oct. 5, 1960), with a positive transparency feature.** Adds Alaska/Hawaii Omnibus Act functions, following EO 10759, EO 10790, EO 10836, and EO 10852. Notable beyond mere confirmation: the new paragraph governing Hawaii-report coordination procedures affirmatively requires that those procedures "shall be published in the Federal Register" — the first instance in this stream of an EO 10530 amendment itself imposing a transparency obligation on the delegated function it creates, rather than simply adding the function silently.

**Major consolidation, paired findings: EO 10893 (Mutual Security) and EO 10900 (Food for Peace) close out the two largest delegation chains tracked in this batch.** EO 10893 (Nov. 8, 1960) supersedes the entire EO 10575 foreign-aid chain (EO 10575, 10610, 10625, 10663, 10742, parts of 10822); EO 10900 (Jan. 5, 1961) supersedes the entire EO 10560 Food for Peace chain (EO 10560, 10685, 10708, 10746, Sections 1-2 of 10799, 10827, 10884) — the same chain diffed twice earlier in this batch (via EO 10746/10799 and again via EO 10884) using the actual claude-church-bells-v1 EO 10560 baseline. Both consolidations score in Flag 3 CRITICAL territory for sheer delegation density, comparable to EO 10480 (DPA) and EO 10773 (civil defense) earlier in this batch. But both also deliberately preserve, rather than discard, the specific positive check each chain had already been credited with: EO 10893 Sec. 110(4) continues reserving to the President the function of "determining any provision of law to be disregarded" (first flagged at EO 10822), and EO 10900 Sec. 5 continues reserving the Sec. 108 congressional-reporting function to the President (first flagged at EO 10560 itself, and unchanged across six amendments and seven years). The pattern across this batch's major consolidations is consistent: administrative concentration and a preserved presidential/congressional check are not in tension — the Eisenhower administration's consolidation orders repeatedly chose to concentrate operational authority while keeping the single most consequential discretionary or reporting function anchored at the top.

**Emergency-authority scope creep, a clean paired example: EO 10896 and EO 10905 use the identical Proclamation 2914/Trading with Enemy Act chain six weeks apart, in opposite directions.** EO 10896 (Nov. 29, 1960) revokes and simplifies provisions of the 1933 gold-regulation order (EO 6260); EO 10905 (Jan. 14, 1961) uses the same authority chain to add a substantial new prohibition — barring U.S. persons from acquiring or holding gold coin, certificates, or bullion located outside the United States without Treasury license. Both orders rely on the same emergency (Proclamation 2914, Dec. 1950) as live legal predicate, now more than a decade old, with no textual reassessment in either order of whether the emergency persists. This is the clearest evidence in the batch that a long-unreviewed emergency proclamation is not merely inertia — it remains fully available to justify brand-new, more restrictive action just as easily as routine housekeeping, with no additional occasion for legislative or public reconsideration triggered by the passage of time or the scale of the new obligation being imposed.

**Positive counter-pattern, appropriate as a closing note: EO 10913 (the batch's final order) is an explicit anti-cannibalization mechanism.** Where several multi-agency frameworks earlier in this batch left overlapping-jurisdiction risk unaddressed, EO 10913 (Jan. 18, 1961) requires Agriculture, Interior, the Army, and TVA to affirmatively request one another's views on the feasibility of achieving overlapping water-resource objectives *before* committing to a planning course, and specifically before any commitment is made to local interests (Sec. 4(e)). A genuinely well-designed ex ante duplication-avoidance requirement, and a fitting structurally-clean bookend two days before the administration ended.

**Also worth noting: EO 10895's Sec. 2 is a rare genuine self-review sunset.** Most standing delegations in this batch carry no built-in reassessment obligation. EO 10895 (Nov. 25, 1960) requires the Secretary of Defense to review the continuing need for the order "at least annually" and recommend revocation when advisable — an actual recurring review duty, not just a hypothetical availability of repeal.

## Operational note (not a substantive finding)

The live `data/eo_coding.db` file, when accessed through the mounted project folder, does not tolerate SQLite's default rollback-journal lifecycle: the filesystem blocks file *deletion* (consistent with this being a version-controlled project folder), and SQLite's default `journal_mode=DELETE` needs to delete its `-journal` file on every commit. The first write attempt in this session left a hot journal behind and made the database briefly unreadable (`disk I/O error` on `PRAGMA integrity_check`). Recovered by truncating the journal file to zero bytes (truncation is permitted; deletion is not) and switching to `PRAGMA journal_mode=PERSIST` for all subsequent writes, which reuses the journal file in place instead of deleting it. Confirmed no data loss: `claude-church-bells-v1`'s 306 existing records were intact throughout. **Any future session writing to this database from this mounted environment should set `PRAGMA journal_mode=PERSIST` before the first write, or writes will fail.**
